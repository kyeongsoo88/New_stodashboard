# 대시보드 데이터 관리 가이드

## 📋 CSV 파일 구조

`dashboard-sample.csv` 파일은 대시보드 탭의 모든 데이터를 월별 컬럼으로 관리합니다.

### 파일 형식
- **인코딩**: UTF-8 BOM (엑셀에서 한글이 깨지지 않도록 설정됨)
- **구조**: 첫 번째 열은 데이터 키, 이후 열은 월별 데이터 (Jan-25, Feb-25, ... Oct-25)

### 데이터 키 명명 규칙
```
카드_[카드명]_[항목명]_[상세항목]_[값타입]
```

예시:
- `카드_실판매출_값`: 실판매출 카드의 메인 값
- `카드_실판매출_채널_US홀세일_값`: 채널별 상세보기 > US홀세일의 값
- `카드_영업비_인건비_급여_값`: 영업비 상세보기 > 인건비 > 급여의 값

### 포함된 데이터

#### 1. 실판매출 카드
- 메인 값, YOY, 매출비중
- 채널별 상세보기: US홀세일, US EC, EU EC, 라이선스
- 아이템별 상세보기: 25FW, 25SS, FW과시즌, SS과시즌, CORE

#### 2. 영업이익 카드
- 메인 값, YOY, 이익률
- 채널별 직접이익[이익율]: US홀세일, US EC, EU EC, 라이선스
- 직접이익 YTD 상세보기: US홀세일, US EC, EU EC, 라이선스

#### 3. 영업비 카드
- 메인 값, YOY, 영업비율
- 영업비 상세보기: 인건비(급여, 보험복지, 급여충당금), 광고선전비, 지급수수료, 식대출장비, 임차료, 샘플비개발비, 감가상각비, 기타비용

#### 4. 할인율 카드
- 메인 값, YOY
- 채널별 상세보기: US홀세일, US EC, EU EC, 라이선스
- US EC 할인율 상세보기: 25FW, 25SS, FW과시즌, SS과시즌, CORE

#### 5. 기타 카드
- 당시즌 판매율, 당시즌 M/U, 기말재고, 인원수

## 🔄 월별 데이터 관리 방법

### 방법 1: 단일 CSV 파일 (권장) ✅
**파일**: `dashboard-2025.csv` (1개 파일)

**장점**:
- 파일 관리가 쉬움 (1개 파일만 수정)
- 엑셀에서 월별 컬럼을 한눈에 비교 가능
- 월별 데이터 추가/삭제가 용이
- CSV 파싱 로직이 단순함

**단점**:
- 파일이 커질 수 있음 (하지만 현실적으로 문제 없음)
- 한 번에 모든 월 데이터를 로드해야 함 (하지만 필요한 월만 필터링 가능)

**사용 방법**:
1. `dashboard-2025.csv` 파일을 엑셀에서 열기
2. 조회 기준으로 선택한 월의 컬럼만 수정
3. 저장

### 방법 2: 월별 CSV 파일
**파일**: `dashboard-2025-01.csv`, `dashboard-2025-02.csv`, ... (10개 파일)

**장점**:
- 월별로 독립적인 파일 관리
- 필요한 월만 로드 가능

**단점**:
- 파일이 많아 관리가 복잡함
- 월별 데이터 비교 시 여러 파일을 열어야 함
- 파일 생성/삭제 관리 필요

## 💡 추천 방법

**방법 1 (단일 CSV 파일)을 권장합니다.**

이유:
1. 엑셀에서 월별 컬럼을 나란히 비교하며 입력/수정하기 쉬움
2. 파일 관리가 단순함 (1개 파일만)
3. 데이터 구조 변경 시 한 곳만 수정하면 됨
4. 조회 기준 드롭다운으로 필요한 월의 컬럼만 읽어오면 되므로 성능 문제 없음

## 📝 데이터 수정 방법

1. `public/data/dashboard-sample.csv` 파일을 엑셀에서 열기
2. 수정하려는 데이터의 행을 찾기
3. 해당 월의 컬럼에 값 입력
4. 저장 (UTF-8 BOM 형식 유지)

## ⚠️ 주의사항

1. **인코딩**: 파일을 저장할 때 UTF-8 BOM 형식을 유지해야 엑셀에서 한글이 깨지지 않습니다.
2. **숫자 형식**: 천단위 콤마는 자동으로 처리되므로 숫자만 입력하세요 (예: `31755` 또는 `31,755` 모두 가능)
3. **퍼센트**: 퍼센트 값은 `%` 기호 없이 숫자만 입력해도 됩니다 (예: `104` 또는 `104%`)
4. **데이터 키**: 데이터 키 이름을 변경하지 마세요. 코드에서 이 키를 사용하여 데이터를 찾습니다.

## 🔧 향후 구현 시 필요한 작업

코드에서 이 CSV 파일을 읽어와서 대시보드에 표시하도록 구현하면:
- 조회 기준 드롭다운에서 선택한 월의 컬럼만 읽어오기
- 각 데이터 키를 대시보드 카드의 props로 매핑하기
